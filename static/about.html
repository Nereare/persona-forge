About

This application is inspired by Dwarf Fortress’ thoughts and preferences section, which contains a plethora of information about each and every character.

Inspired by such a level of complexity, I decided to create a default format under which one can generate a somewhat equally complex description for one’s persona.

About the Data

No data fed into the application is saved or stored in any manner on the server. This application runs entirely on the client side through JavaScript and AngularJS.

About Cookies

This application does not use any cookies for any of its functionalities.
